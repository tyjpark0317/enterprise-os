## Agent and Hook Reorganization

Audit all agents, hooks, and settings. Optimize structure, reduce duplication, improve maintainability.

**Scope**: agents/ + hooks/ + hook configuration
**Approach**: Audit -> Deep Research -> Reference Map -> Agent-Skill Mapping -> Propose -> Execute

### Steps

0. **Pre-Flight Validation** — load protection rules
1. **Audit** — full agent + hook inventory
2. **Deep Research** — AI engineering best practices
3. **Reference Map** — full system cross-reference map
4. **Agent-Skill Mapping** — optimal skill connections per agent
5. **Propose** — reorganization plan (user approval required)
6. **Execute** — apply approved changes
7. **Verify** — all references resolve, hooks run, JSON valid

## Checklist

- [ ] Pre-flight validation
- [ ] Audit agents + hooks
- [ ] Deep research
- [ ] Reference map
- [ ] Agent-skill mapping
- [ ] Propose changes (user approval)
- [ ] Execute changes
- [ ] Verify
